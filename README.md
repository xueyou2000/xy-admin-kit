`xy-admin-kit`提供通用工具, 比如数组工具, 字符串工具等

## 安装

在对应项目中运行:

```sh
# yarn
yarn add wallet-generate

# npm
npm add -d wallet-generate
```

## 内容

-   `http` 一个简单实现的网络请求工具
-   `lib`
    -   `wx-sdk` 微信 Sdk 的封装
-   `security` jwt 认证封装
-   `socket` WebSocket 封装
-   `utils`
    -   `array-utils`
    -   `date-utils`
    -   `injector-utils`
    -   `number-utils`
    -   `object-utils`
    -   `string-utils`
